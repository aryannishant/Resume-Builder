About The Application

RESUME BUILDER An application that lets you create simple resume on the go. It has a simple UI which helps the user to fill their required details and choose the template they are comfortable in. The application gives the power to users to download their resume in JPEG / PNG or PDF format thus giving enough options on providing the most efficient resume. The tech stacks include HTML5 , CSS3 & Vanilla Javascript

RULES OF RESUME

  • The text can be edited using HTML i.e bold, italics , underline , strike etc.
  • Hyperlink can be added using HTML as well i.e "a" tag
  • To stop the abuse of resume we have limited the skill to 10 inputs and others to 5 inputs only
  • First input value must be filled, otherwise the section will be skipped
  • Do not worry about the wrong input value, you can edit this value after moving to next page
  • Wev highly recommend you to keep input as brief as possible
  • Inputs will be ordered do not add bullet or serial number
